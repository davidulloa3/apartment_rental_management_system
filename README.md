# Apartment Rental Management System

## Project Overview

The Apartment Rental Management System is a PostgreSQL database project designed to manage apartment rental information. The system stores and organizes data for tenants, properties, units, leases, payments, and maintenance requests.

This project demonstrates how a relational database can be used to support real property management tasks, including tracking active leases, recording rent payments, identifying late payments, and managing open maintenance requests.
